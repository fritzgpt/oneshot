# CLI for Oneshot AI coding

## General Usage

```bash
# basic
./oneshot.py shoot \
  --pattern|-p my-pattern \
  --pattern-dir=dir-path \
  --env-file=file-path \
  --with-tools|-t \
  --output-to-disk|-o \
  --model|-m \
  [Specific User Request]

# generate patterns
./oneshot.py generate \
  --template-dir|-t=dir-path
  --template-dir|-t=dir-path-2
```
